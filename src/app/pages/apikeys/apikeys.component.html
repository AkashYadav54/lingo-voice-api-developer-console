Create and manage API Keys

Remember to save your API Keys in a safe spot. For security purposes, secrets are not recoverable.

API Keys act like a password for API Authentication. Insert the secret from your API Key into your request like so:

Authorization: Token <YOUR_SECRET>

Your secret should look like a 40 character alphanumeric.

Here you can add best practices for managing API keys.

If you forgot your API secret, create a new API key to replace it.
